# Richbee

The template uses:
- [Gulp](http://gulpjs.com/)
- [Browser-sync](https://www.browsersync.io/) - automatic restart when changing the code page
- [Scss](http://sass-lang.com/)
- Merge and compress CSS and JavaScript


## Get started
Install packages:
```bash
npm install
```

And run command:
```bash
gulp
```
